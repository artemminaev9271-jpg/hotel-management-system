# Система бронирования отелей

Полнофункциональное десктопное приложение для поиска и бронирования отелей с веб-сервером на FastAPI и графическим интерфейсом на CustomTkinter.

# Описание

Приложение представляет собой систему управления бронированиями отелей с разделением ролей пользователей. Клиенты могут искать отели, бронировать номера и оставлять отзывы. Администраторы имеют доступ к панели управления для добавления новых отелей и номеров.

# Основные возможности:

- Регистрация и авторизация пользователей
- Поиск отелей по городу
- Система рейтингов и отзывов
- Бронирование номеров с выбором дат
- Проверка конфликтов бронирований
- Административная панель для управления контентом
- История бронирований пользователя

# Технологии

# Backend:
- **FastAPI** - веб-фреймворк для REST API
- **SQLAlchemy** - ORM для работы с базой данных
- **SQLite** - реляционная база данных
- **Pydantic** - валидация данных

# Frontend:
- **CustomTkinter** - современная библиотека для GUI
- **Pillow** - обработка изображений
- **Requests** - HTTP-клиент для взаимодействия с API

# Testing:
- **Pytest** - фреймворк для тестирования
- **HTTPX** - асинхронный HTTP-клиент для тестов

# Установка

# Требования:
- Python 3.10 или выше
- pip

# Шаги установки:

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/artemminaev9271-jpg/hotel-management-system.git
cd hotel-management-system
```
2. **Создайте виртуальное окружение:**
```bash
python -m venv venv
```
3. **Активируйте виртуальное окружение:**
```bash
venv\Scripts\activate
```
4. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

# Запуск приложения:

1. **Запуск сервера:**
```bash
cd server
uvicorn main:app --reload
```
2. **Запуск клиентского приложения:**
```bash
cd client
python main.py
```

# Тестирование

1. **Запуск модульных тестов:**
```bash
pytest tests/test_main.py -v --tb=short
```
2. **Запуск с покрытием кода:**
```bash
pytest tests/test_main.py --cov=server --cov-report=html
```
